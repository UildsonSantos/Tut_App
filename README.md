## Run this project

### 1 - Clone this repository
```
git clone https://github.com/UildsonSantos/Tut_App.git
```
### 2 - Open the project in android studio [Android](https://developer.android.com/studio/install)

### 3 - Activate an emulator or connect your android smartphone


### 4 - Type in android studio terminal:

```
flutter run --no-sound-null-safety
```
